# LoLScan User Guide / Руководство Пользователя

## English

### What is LoLScan?
LoLScan helps monitor your PC activity and alerts and blocks you when suspicious command behavior is detected.

### Installation
1. Run `LoLScan-Setup-x64.exe`.
2. Follow the installer steps.
3. When installation is finished, open LoLScan.

### First Launch
1. Wait until the app finishes loading.
2. In the **Protection** screen, turn protection **ON**.
3. In **Settings**, check that **Autostart** is enabled (it is enabled by default).

### Main Screens
- **Protection**: turn protection on/off and see current status.
- **Detections**: view blocked or suspicious activity.
- **Logs**: view the app activity history.
- **Settings**: manage startup behavior.

### Daily Use
1. Keep LoLScan running in the background.
2. If the window is closed, it continues from the tray icon.
3. Open it anytime from the tray icon to check detections and logs.

### If You See Alerts
1. Open **Detections**.
2. Review what was blocked.
3. If needed, use **Logs** for more context.

### Uninstall
1. Open Windows **Apps** settings.
2. Find **LoLScan** and choose **Uninstall**.

## Русский

### Что такое LoLScan?
LoLScan помогает отслеживать активность на ПК и предупреждает и блокирует о подозрительном поведении команд.

### Установка
1. Запустите `LoLScan-Setup-x64.exe`.
2. Выполните шаги установщика.
3. После установки откройте LoLScan.

### Первый запуск
1. Дождитесь завершения загрузки приложения.
2. На экране **Защита** включите защиту (**ON**).
3. В **Настройки** проверьте, что **Запускать с Windows** включен (по умолчанию он уже включен).

### Основные разделы
- **Защита**: включение/выключение защиты и текущий статус.
- **Обнаружения**: просмотр заблокированной или подозрительной активности.
- **Логи**: история работы приложения.
- **Настройки**: управление запуском при входе в систему.

### Ежедневное использование
1. Оставляйте LoLScan работать в фоне.
2. При закрытии окна приложение продолжает работать через иконку в трее.
3. Открывайте приложение из трея, чтобы проверять срабатывания и журнал.

### Если появились предупреждения
1. Откройте **Обнаружения**.
2. Посмотрите, что было заблокировано.
3. При необходимости проверьте **Логи** для дополнительного контекста.

### Удаление
1. Откройте настройки приложений.
2. Найдите **LoLScan** и нажмите **Удалить**.
